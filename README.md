# Bundling Server Example Project

This project contains multiple packages for the dynamic bundling server example:

- bundling-server
- drum-samples
- drum-sequencer
- basic-button
- advanced-button
- sdk-release

## Installation

```bash
npx lerna bootstrap
```

## Usage

```bash
cd packages/bundling-server
npm start
```
